# 사랑의병원 감염관리 교육 게임 업그레이드

포함 내용
- 시작 화면에 마스코트 이미지 적용 + 부드러운 움직임
- 게임 내 플레이어에 캐릭터 스프라이트 적용 (대기/이동/공격)
- 버블이 아이템 획득 시 활성화되어 보조 공격
- 적 스프라이트 적용
- Stage 1~3 교육형 가이드 추가
- 각 스테이지마다 어떤 감염병이 나오는지, 어떤 도구가 예방에 중요한지 설명

배포 방법
1. 이 ZIP 압축 해제
2. 기존 Vercel 프로젝트 파일을 전부 교체
3. 다시 배포

참고
- 사용 이미지 파일은 assets 폴더에 포함되어 있습니다.
- 현재 구조는 순수 HTML/CSS/JS 이므로 바로 배포 가능합니다.
